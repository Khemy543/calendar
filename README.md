# Calendarity

Calendarity is a modern and fully customizable calendar application built with Vue.js, Sass, and Pinia. This application is designed to help you manage your events efficiently with an intuitive interface and powerful features.

## Features

- **Monthly, Weekly, and Daily Views**: Easily switch between different calendar views to see your events.
- **Event Management**: Create, update, and delete events with a user-friendly interface.
- **Responsive Design**: Optimized for both desktop and mobile devices to ensure a seamless experience.
- **Customizable Themes**: Personalize the appearance of the calendar with various themes.
- **Drag and Drop**: Rearrange events effortlessly with drag-and-drop functionality.
- **Notifications and Reminders**: Set reminders for your events to stay organized.
- **Search and Filter**: Quickly find events with the search and filter features.

## Installation

Follow these steps to set up Calendarity locally:

1. **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/Calendarity.git
    cd Calendarity
    ```

2. **Install dependencies**
    ```bash
    npm install
    ```

3. **Run the development server**
    ```bash
    npm run dev
    ```

4. **Build for production**
    ```bash
    npm run build
    ```

## Usage

To start using Calendarity, simply run the development server and open your browser to `http://localhost:8080`. You can navigate through different calendar views, create and manage events, and customize the application settings to suit your preferences.
